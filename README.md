# Автоматизация документооборота для ООО «Экспертиза П-Б»

Дипломный проект по автоматизации процесса подготовки экспертных заключений и отчётных документов.

## Функциональность
- Авторизация с ролевой моделью (администратор, руководитель, эксперт, специалист)
- Создание и ведение проектов
- Загрузка результатов обследования (с новыми полями: давление, температура, материал и т.д.)
- Автоматическое формирование документов по шаблонам (акт, отчёт, заключение)
- Согласование и утверждение документов
- Генерация PDF и сохранение в архиве
- Отчёты и журнал аудита
- Резервное копирование базы данных

## Технологии
- Python 3.8+
- Tkinter (GUI)
- MySQL 8.0
- docxtpl, docx2pdf

## Установка и запуск
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Octik5/expertise-automation.git
   cd expertise-automation
2. Создайте виртуальное окружение (рекомендуется)
   ```bash
   python -m venv venv
   source venv/bin/activate   # Linux/Mac
   venv\Scripts\activate      # Windows
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
4. Создайте базу данных MySQL с именем expertise_db_db и выполните скрипт database/schema.sql.
5. Скопируйте config_example.py в config.py и укажите свои параметры подключения.
6. Запустите программу:
   ```bash
   python mainn.py
